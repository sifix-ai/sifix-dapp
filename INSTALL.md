# Instructions to Install Required Dependencies

The project needs these additional dependencies:

```bash
npm install framer-motion class-variance-authority @radix-ui/react-slot
```

## If WSL is Required:

Open WSL terminal and run:

```bash
cd /mnt/d/web3/sifix-dapp
npm install framer-motion class-variance-authority @radix-ui/react-slot
```

## Or Install Directly in Windows:

```bash
cd D:\web3\sifix-dapp
npm install framer-motion class-variance-authority @radix-ui/react-slot
```
